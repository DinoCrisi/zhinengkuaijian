# 爆款复刻提示词系统升级说明 (v2.5.0)

## 升级概述

本次升级采用了专业的短视频分析提示词系统，大幅提升了脚本生成和首帧图片生成的质量。

## 主要改进

### 1. 专业的分析框架

新提示词基于短视频算法与内容营销的专家级分析逻辑，包含：
- 精准的标签判定规则
- 详细的思维链处理步骤
- 专业的输出定义规范

### 2. 精准的标签体系

#### 一级标签（必须唯一）
- **hook（钩子）**：视频前0-5秒，吸引注意力，强留人语气
- **selling_point（卖点）**：产品介绍，功能描述，价格优势，解决痛点
- **proof（证明）**：效果展示，对比镜头，第三方检测，使用演示
- **cta（转化）**：引导购买，点击下方，下单链接

#### 二级标签
- **信息密度**：高密度/中密度/低密度
- **视觉语言**：人脸直出/产品特写/行为演示/对比镜头/场景空镜/情绪反应

### 3. 详细的首帧提示词格式

**新格式**：构图 + 光影 + 色彩 + 神态 + 主体 + 场景 + 质量词

**详细要求**：
- **构图**：景别、拍摄角度、主体位置、画面比例
- **光影**：光源方向、光质、明暗对比、光影效果
- **色彩**：主色调、色彩饱和度、色彩对比、色彩氛围
- **神态**：人物表情、姿态、动作状态（如有）
- **主体**：商品的外观、材质、纹理、形状、大小、包装风格、展示方式
- **场景**：背景环境、地点、环境元素、氛围营造
- **质量词**：4K超高清、照片级真实感、细节丰富、专业商业摄影

**长度要求**：80-150 字（比之前的50-100字更详细）

**示例**：
```
特写镜头，居中构图，一只白皙细腻的手背占据画面主要位置，[产品名]的滴管从上方滴下一滴透明液体，液体呈球形悬停状态。光线明亮柔和，来自左上方的散射光，强调手背的细腻质感和液体的透明感。色彩以白色和透明为主，背景为纯净的浅灰色。手背皮肤纹理清晰可见，滴管金属质感反光，液体表面有轻微的光泽。4K超高清，照片级真实感，细节丰富，专业商业摄影。
```

### 4. 更详细的分镜脚本内容

每个分镜的 script_content 现在包含：
- **主体描述**：核心人物/物品的外观、衣着、材质、颜色
- **场景环境**：背景细节、地点、空间感
- **运镜构图**：景别、拍摄角度、镜头运动
- **光影效果**：光线类型、色调
- **动作动效**：主体动作、速度、运动轨迹
- **听觉事件**：画面内的拟音和环境音（不包含BGM和旁白）

- ✅ AI 图像生成质量更高

## 使用建议

1. **上传商品图片**：建议上传 3-5 张高质量的商品图片，AI 会将其作为参考图
2. **填写详细卖点**：每个卖点都会在对应的分镜中重点体现
3. **查看脚本详情**：生成后仔细查看每个分镜的配音文案、画面描述和首帧提示词
4. **确认后生成首帧**：确认脚本无误后，进入首帧生成阶段

## 技术细节

- **修改内容**：完全重构了 `buildScriptReplicationPrompt` 函数
- **版本号**：v2.5.0

## 注意事项

- 首帧提示词严禁包含字幕或文字元素（避免影响生成效果）
- 每个分镜必须且只能有一个一级标签（hook/selling_point/proof/cta）
- 输出格式必须是纯 JSON，不包含 markdown 标记
